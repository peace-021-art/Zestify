# Zestify 🍋

Zestify is a modern, high-end e-commerce grocery storefront designed with a sleek, minimalist, and premium aesthetic. Built with a focus on user experience, it features dynamic interactive elements without relying on heavy front-end frameworks.

## 🌟 Features

- **Modern Mega-Menu:** Features an Amazon-style sticky dropdown menu that bridges diagonal mouse paths to prevent frustrating accidental menu closures.
- **Dynamic Shopping Cart:** An interactive sliding drawer for reviewing cart items, adding to cart, and tracking orders.
- **Authentication Flow:** Built-in modal components for seamless User Sign-In and Account Registration without navigating away from the page.
- **Interactive Carousels & Sliders:** Engaging hero banners and category sliders built with Vanilla JavaScript.
- **Responsive Layout:** Carefully designed to look stunning on desktops, tablets, and mobile devices.
- **Premium Design System:** Uses a scandi-inspired neutral color palette (`#f9f8f6`, `#efe9e3`, `#d9cfc7`) accented with vibrant food imagery to create a premium, Apple-like shopping experience.

## 🛠️ Tech Stack

Zestify is built to be extremely lightweight and fast:
- **HTML5:** Semantic, accessible structure.
- **Tailwind CSS:** For rapid, utility-first styling and complex animations (including custom hover delays and transitions).
- **Vanilla JavaScript:** For DOM manipulation, state management (cart and auth), and interactive components—keeping the application dependency-free.
- **FontAwesome:** For crisp, scalable iconography.

## 🚀 Getting Started

Since Zestify is built with pure HTML, CSS, and JS, there is no complex build step required!

1. Clone the repository:
   ```bash
   git clone https://github.com/peace-021-art/Zestify.git
   ```
2. Open the project folder.
3. Simply double-click `index.html` to open it in your web browser, or use an extension like **Live Server** in VS Code for hot-reloading.

## 🎨 Design Philosophy

Zestify moves away from the traditional, cluttered grocery store layout. By removing overly "marketing-heavy" language and focusing on high-quality photography, soft depths, and smooth micro-interactions, Zestify aims to provide a calming and luxurious online shopping experience.
